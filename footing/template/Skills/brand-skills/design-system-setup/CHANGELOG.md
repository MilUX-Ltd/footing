# Changelog — design-system-setup

## 1.0.0 — 2026-06-18

First release. Generates a complete, token-based design system from a short interview:

- Canonical token file (`css/colors_and_type.css`): brand palette with derived scales, a six-stop chart palette, type scale, spacing, radii, borders, elevation.
- Stylesheet entry point (`css/styles.css`) and semantic element styles.
- Generated showcase `index.html`: colour swatches, type specimens, components, voice.
- Written `BRAND-GUIDANCE.md` from the founder's voice rules.
- Starter templates: A4 report, 16:9 slide, letterhead.
- Dependency-free Node scaffolder; derived chart accents clamped to a tasteful saturation so they stay harmonious for any input.

Structure follows the MilUX design system; values are the founder's own.
