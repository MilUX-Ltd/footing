Vault Viewer · v1.1.0

Read a folder of Markdown notes like a website

A single-file reader for any folder of Markdown files. No install, no cloud, nothing leaves your machine. Browse it with wikilinks, search and backlinks all working. It understands Obsidian-style links and embeds, but Obsidian isn't required.

Uses the browser's folder picker (File System Access API). Works in Chrome, Edge, Brave and other Chromium browsers. Files are read locally and never uploaded.
